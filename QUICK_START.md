# دليل البدء السريع - SIM-NET

## 🚀 ابدأ في 3 خطوات!

### الخطوة 1: تثبيت Python
تأكد من تثبيت Python 3.8 أو أحدث:
```bash
python --version
```

### الخطوة 2: تشغيل النظام

**Windows:**
```bash
python launcher.py
```

**Linux/Mac:**
```bash
python3 launcher.py
```

### الخطوة 3: فتح المتصفح
افتح المتصفح على:
```
http://localhost:8080
```

سجل الدخول بـ:
- **Username:** admin
- **Password:** admin123
